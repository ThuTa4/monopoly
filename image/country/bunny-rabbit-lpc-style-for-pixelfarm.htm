Included is a bunny rabbit, commissed by tebruno99 for his (awesome) PixelFarm project.  It's made to be compatible with this (awesome) set of farm animals by Daneeklu http://opengameart.org/content/lpc-style-farm-animals

Included are four directional hopping and grass-munching.

Not much else to say, except that you shoudl be sure to check out tebruno99's PixelFarm project here: https://bitbucket.org/tebruno99/pixelfarm
